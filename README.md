# 藏经阁 — Self Knowledge Database

> glm-扫地僧 守阁之所。提纯后的技术知识库，面向快速检索。
> 知识入库前经过多源交叉验证，标注置信度与信源。

## 知识库结构

```
├── KB-LINUX-DRM/          # Linux DRM/KMS 子系统
├── KB-LINUX-GPU-DRIVER/   # Linux GPU 驱动开发
├── KB-GPU-ARCH/           # GPU 架构
├── KB-GFX-API/            # 图形 API (Vulkan / DX12 / OpenGL)
├── KB-AI-LLM/             # 大语言模型
├── KB-AI-INFRA/           # AI 基础设施（训练/推理/芯片）
└── KB-AI-AGENT/           # AI Agent 框架与工具链
```

## 入库规范

| 维度 | 规则 |
|------|------|
| 信源优先级 | 内核源码 > 官方文档 > 权威社区 > 技术博客 |
| 入库门槛 | 多源交叉验证，不通过的不入库 |
| 标注规范 | [置信度 1-5] [最后验证日期] [主要信源] |
| 条目结构 | 概述 → 核心流程/架构 → 关键数据结构/API → 常见陷阱 → 参考链接 |
| 更新策略 | 禁止覆盖，新信息作为版本迭代追加 |
